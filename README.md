# AI Resume Builder

## Project Description
An intelligent resume generation system that creates customized, ATS-friendly resumes based on user inputs.

## Setup Instructions
1. Open `index.html` in any modern web browser (Chrome, Edge, Firefox).
2. Enter your Google Gemini API Key in the code where indicated.
3. Fill in your details and click "Generate" to see AI suggestions.

## Features
- AI Content Generation (Gemini API)
- Real-time Preview
- ATS Score Checking
- PDF Export (html2pdf)
- Local Storage (Auto-save)

## Technical Details
- Frontend: HTML5, Tailwind CSS
- Logic: Vanilla JavaScript
- API: Google Gemini 1.5 Flash
